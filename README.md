
# Open Source Match

A platform connecting junior developers with beginner-friendly open-source projects to gain real-world experience.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:5173](http://localhost:5173) in your browser.

4. see it here [https://open-source-match-beta.vercel.app/] (https://open-source-match-beta.vercel.app/)
